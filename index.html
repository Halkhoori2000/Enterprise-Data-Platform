About this model

An interactive, fully-generalized model of a real enterprise banking data platform — how dozens of source systems, event streams, and years of files become governed, queryable data products. It reads like a real reference architecture, and you choose how deep to go.

  • L0–L4 set the depth — L0 overview, L1 the full estate, then L2–L4 go inside a clicked piece (keys 0–4 jump levels).
  • Click any piece to open its detail panel; the security & governance rails run underneath everything.
  • ▶ pieces have a live demo — Kafka's commit log, Delta's transaction log, a Spark job, and a record refining Bronze → Silver → Gold.
  • Drag to pan, scroll to zoom, fit to reset — or take the guided tour from the “A record’s journey” card.
🔒 Ideas, architecture, and technology only — no institution names, internal systems, configurations, or data appear anywhere.
